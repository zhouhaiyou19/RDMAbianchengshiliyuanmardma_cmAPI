# RDMA编程实例源码（rdma_cm API）

## 简介

本仓库提供了一个RDMA（远程直接内存访问）编程实例的源码，使用的是rdma_cm API。RDMA是一种高性能网络通信技术，允许数据在网络中的不同节点之间直接传输，而无需通过CPU的介入，从而显著提高了数据传输的效率和性能。
